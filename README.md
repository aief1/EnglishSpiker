# EnglishSpiker

一个 Windows 桌面小工具：复制英语单词后自动朗读，并显示中文意思。

## 使用方法

1. 双击 `start_english_reader.bat`。
2. 程序会显示一个小浮窗，默认始终置顶。
3. 复制一个英语单词，程序会自动朗读并显示中文释义。
4. 按默认快捷键 `Ctrl+Alt+R` 可以重读剪贴板里的单词。
5. 点窗口右上角的 `×` 不会退出程序，只会隐藏到屏幕右下角托盘。

## 说明

- 朗读使用 Windows 自带语音功能，不需要额外安装 Python 包。
- 设置都在托盘图标右键菜单里：显示/隐藏、自动朗读、自动释义、置顶、声音、语速、快捷键、退出。
- 声音好不好听取决于 Windows 已安装的语音；可以在托盘右键菜单里切换。
- 中文释义优先使用在线词典接口，网络不可用时会给出提示。

## 命令行启动

```powershell
py english_reader_widget.py
```
