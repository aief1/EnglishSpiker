# 英语单词朗读工具

一个 Windows 桌面小工具：把英语单词复制到输入框里，它会朗读这个单词，并显示中文意思。

## 使用方法

1. 双击 `start_english_reader.bat`。
2. 把英语单词粘贴到输入框里，或者点“读剪贴板”。
3. 程序会自动朗读并查询中文释义。

## 说明

- 朗读使用 Windows 自带的语音功能，不需要额外安装语音包。
- 中文释义优先使用在线词典接口；如果网络不可用，会提示查询失败。
- 如果你只想手动操作，可以取消勾选“输入后自动朗读和查询”，再点击“朗读”或“查中文意思”。

## 命令行启动

```powershell
py english_reader.py
```
